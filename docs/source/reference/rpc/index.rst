RPC
===

This section of the documentation contains information on the RPC protocol,
the involving parts and classes, and how they communicate. This section isn't
intended for the users of Clio, but for core Clio developers, as a reference
for those who are interested, or for those who want to use the Clio RPC stack
independently.
